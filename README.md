# Distribution-Shift-AIES-2025
This repository contains the code for all experiments presented in our paper:
**'When the Past Misleads: Rethinking Training Data Expansion Under Temporal Distribution Shifts'**

## Citation
If you find our paper or code useful in your research, please cite us: "Yao, Chengyuan, Yunxuan Tang, Christopher Brooks, Rene F. Kizilcec, and Renzhe Yu. "When the Past Misleads: Rethinking Training Data Expansion Under Temporal Distribution Shifts." In Proceedings of the AAAI/ACM Conference on AI, Ethics, and Society, vol. 8, no. 3, pp. 2760-2771. 2025."
